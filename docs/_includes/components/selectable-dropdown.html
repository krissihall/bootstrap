Selectable Dropdowns

This is an extension of the Dropdowns section. This is styled using select-control behavior where the selected option populates the dropdown toggle, rather than a static <button> to toggle the menu.

This is an extension of the dropdown as described above. In order to use this variation where the label text is pushed up and the selected value is displayed, you can use the same syntax and just add the dropdown-selection class do the dropdown container as well as provide a label in the data-dropdown-label attribute.
